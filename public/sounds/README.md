# Sound Files

This directory contains audio feedback files for the ticket scanner:

- **success.mp3** - Played when a valid ticket is scanned
- **error.mp3** - Played when an invalid ticket is detected

## Adding Sound Files

You can add free sound files from these sources:

1. **Freesound.org** - https://freesound.org/
   - Search for "success beep" or "error sound"
   - Download CC0 (public domain) sounds

2. **Zapsplat** - https://www.zapsplat.com/
   - Free sound effects library
   - Look in UI/Interface sounds

3. **Mixkit** - https://mixkit.co/free-sound-effects/
   - Free game and UI sound effects

## Recommended Sounds

- **Success**: Short, positive beep (200-500ms)
- **Error**: Low-pitched buzz or negative tone (200-500ms)

Place the files here as:
- `public/sounds/success.mp3`
- `public/sounds/error.mp3`
